\doxysubsection{Detailed Description}
A decorator class for enhancing the functionality of citizen objects. 

This class implements the \mbox{\hyperlink{classCitizenInterface}{Citizen\+Interface}} and wraps around a \mbox{\hyperlink{classCitizenInterface}{Citizen\+Interface}} instance to provide additional functionalities such as salary management, employment status, and property ownership tracking.
