\doxysubsection{Detailed Description}
A factory class for creating residential buildings. 

This class inherits from \mbox{\hyperlink{classBuildingFactory}{Building\+Factory}} and provides an interface for creating residential building objects. Derived classes must implement the \mbox{\hyperlink{classResidentialFactory_af98fef03d68b3ccda81dcd27e922d680}{create\+Building()}} method to produce specific types of residential buildings.
