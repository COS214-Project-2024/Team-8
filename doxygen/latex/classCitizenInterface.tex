\doxysubsection{Detailed Description}
An abstract base class that defines the interface for \mbox{\hyperlink{classCitizen}{Citizen}} components. 

This interface provides a common structure for all citizen-\/related classes, including functions for tax calculation, salary adjustments, public service requests, and cloning capabilities.
