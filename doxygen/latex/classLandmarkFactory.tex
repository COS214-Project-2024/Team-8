\doxysubsection{Detailed Description}
A factory class for creating landmark buildings. 

This class inherits from \mbox{\hyperlink{classBuildingFactory}{Building\+Factory}} and provides an interface for creating landmark building objects. Derived classes must implement the \mbox{\hyperlink{classLandmarkFactory_af0a8e396c38942baed18814cc7504119}{create\+Building()}} method to produce specific types of landmark buildings.
