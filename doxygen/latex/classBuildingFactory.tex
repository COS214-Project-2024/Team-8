\doxysubsection{Detailed Description}
An abstract factory class for creating Building objects. 

This class defines a factory interface for creating building objects. Derived classes must implement the \mbox{\hyperlink{classBuildingFactory_a553d8ca955c6e4421ea2de7a616fe9e6}{create\+Building()}} method to produce specific types of buildings.
