\doxysubsection{Detailed Description}
A factory class for creating commercial buildings. 

This class inherits from \mbox{\hyperlink{classBuildingFactory}{Building\+Factory}} and provides an interface for creating commercial building objects. Derived classes must implement the \mbox{\hyperlink{classCommercialFactory_a06ff168b783133fad993bc6579fe5a00}{create\+Building()}} method to produce specific types of commercial buildings.
