\doxysubsection{Detailed Description}
A factory class for creating industrial buildings. 

This class inherits from \mbox{\hyperlink{classBuildingFactory}{Building\+Factory}} and provides an interface for creating industrial building objects. Derived classes must implement the \mbox{\hyperlink{classIndustrialFactory_ac1eba9a085eaf6c0a7ea915b6e95f83c}{create\+Building()}} method to produce specific types of industrial buildings.
