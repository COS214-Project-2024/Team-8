\doxysubsection{Detailed Description}
A factory class for creating public service buildings. 

This class inherits from \mbox{\hyperlink{classBuildingFactory}{Building\+Factory}} and provides an interface for creating public service building objects. Derived classes must implement the \mbox{\hyperlink{classPublicServiceFactory_a01bc339cd89079c83630cda53be4a33b}{create\+Building()}} method to produce specific types of public service buildings.
